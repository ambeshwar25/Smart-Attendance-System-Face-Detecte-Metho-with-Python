# Smart-Attendance-System-Face-Detecte-Metho-with-Python
A Python based system that uses face-detection to mark attendance automatically. It captures real-time images, recognizes faces using trained data, and records attendance accurately, reducing manual work and preventing proxy attendance.

Technologies Used Python Libraries- 
OpenCV (for face detection and image processing)
NumPy (for numerical operations)
Face Recognition libraries (for identifying faces)
CSV / Database (for storing attendance records)

Features-
Automatic attendance marking using face recognition
Real-time face detection through webcam
High accuracy and fast processing
Prevents proxy or duplicate attendance
Easy to use and user-friendly interface
Stores attendance records digitally (CSV/database)
Contactless and time-saving system
